# IncidentLite

A serverless full-stack incident tracking demo built with:

- React + TypeScript (Vite)
- Azure Functions (Node.js)
- Azure Static Web Apps
- Cosmos DB (planned)
- Microsoft Entra ID auth (planned)

## Local Development

### Run API
cd apps/api
func start

### Run Web
cd apps/web
npm run dev

## Endpoints

GET /api/health
